# Python Turtle Snake & Monster Game

## Overview
This is a custom variation of the classic Snake game implemented in Python using the `turtle` graphics library. 

## Game Mechanics
Unlike the traditional game, this version introduces enhanced mechanics:
- **Number Consumption:** The player controls the snake using arrow keys to eat randomly generated numbers in sequence.
- **Monster Evasion:** Multiple wandering monsters patrol the screen. The player must strategically navigate the snake to avoid collisions with them.
- **Growth & Win Condition:** The snake extends its tail upon consuming numbers. The game is won when all numbers are collected without being caught by the monsters.

## Tech Stack
- **Language:** Python 3
- **Libraries:** `turtle` (GUI & Event-driven programming), `random`, `functools`, `time`

## How to Play

1. Run the script by executing the following command in your terminal:

`python snake_monster_game.py`

2. Click anywhere on the screen to start the game.
3. Use the **Up, Down, Left, Right** arrow keys to maneuver the snake.
